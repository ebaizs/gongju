# 常用工具网站交互设计

## 核心交互功能

### 1. 工具分类导航（4个标签页）
- **代码工具**：代码混淆、代码格式化、代码压缩
- **文本处理**：文本替换、文本加密、文本对比
- **图像工具**：图像压缩、格式转换、Base64转换
- **网络工具**：URL编码、时间戳转换、JSON格式化

### 2. 代码混淆工具核心交互
**智能检测系统**：
- 自动检测输入代码类型（HTML/JS/CSS混合）
- 实时分析代码结构，识别混合代码
- 自动勾选适用的混淆选项

**混淆功能选项**：
- 移除注释（自动检测并勾选）
- 压缩空白字符（自动检测并勾选）
- 重命名变量（自动检测并勾选）
- 编码字符串（自动检测并勾选）
- 混淆后保持功能完整性

**实时预览**：
- 左侧输入，右侧实时显示混淆结果
- 显示代码压缩率和混淆统计
- 一键复制和下载功能

### 3. 代码格式化工具交互
**多功能格式化**：
- HTML/JS/CSS混合代码格式化
- 自动修复符号错误（括号匹配、分号补全）
- 智能缩进和换行

**文件拆分功能**：
- 自动识别HTML中的CSS和JS代码块
- 一键拆分为独立的HTML、CSS、JS文件
- 自动添加全局调用变量和引用
- 保证拆分后代码可正常运行

**智能注释系统**：
- 逐行注释模式（为每行代码添加解释性注释）
- 按功能块注释（为函数、样式类、逻辑块添加概述注释）
- 支持多种注释风格

### 4. AI编程关键词工具
**智能问答模板**：
- 预设常用编程问题模板
- 关键词快速插入（"用替换法告诉我怎么改"、"拆除代码"、"增加全局变量"等）
- 一键复制AI对话指令

## 用户体验流程
1. 用户进入主页面，看到4个工具分类
2. 点击"代码工具"分类，看到代码混淆工具图标
3. 点击混淆工具图标，进入专业工具界面
4. 粘贴代码，系统自动检测类型并勾选选项
5. 实时预览混淆结果，一键应用
6. 可切换到格式化工具进行进一步处理
7. 所有操作支持撤销和版本对比
